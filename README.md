# Graduation Projects Portfolio

This repository contains a comprehensive, production-grade portfolio of **34 engineering-centric graduation project proposals** designed for final-year Computer Engineering, Software Engineering, and Computer Science students.

Every project proposal is structured as an authoritative technical design specification prepared from the perspective of CTOs and senior software architects.

---

## 🎯 Portfolio Philosophy & Core Principles

1. **Engineering Complexity Over Simple Wrappers:** The primary contribution of every project is software architecture, distributed systems, backend engineering, cloud infrastructure, networking, cybersecurity, or embedded systems — not merely fine-tuning an AI model.
2. **AI as a Bounded Subsystem (10–30% Effort):** For projects incorporating AI, the AI component is bounded to a specific subsystem (forecasting, anomaly scoring, classification feeding a control decision). **If the AI module were removed entirely, every project remains a deployable, valuable engineering product.**
3. **Zero-Placeholder Policy:** Every project README is fully written with complete section depth, including Executive Summary, Problem Statement, System Architecture, Tech Stack, Required Knowledge, Team Role Distribution, Budget (EGP/USD), and Evaluation Metrics.

## 📑 How to Navigate This Portfolio

- **Looking for a specific domain?** Browse the project folders in `projects/` directly or check `DOMAINS.md`.
- **Forming a team and matching roles?** Check `comparison/CareerPaths.md` to match team members with target career tracks.
- **Seeking corporate sponsorship or startup ideas?** Consult `comparison/Sponsorship.md` and `comparison/StartupPotential.md`.
- **Deciding on the best overall project?** Read `comparison/FinalRecommendation.md`.
